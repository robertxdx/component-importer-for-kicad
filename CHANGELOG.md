# Changelog

## 0.1.0 - 2026-05-18

Initial public release.

- Added Windows GUI for importing component ZIP files into KiCad project libraries.
- Added project-local symbol, footprint, 3D model, metadata, source ZIP, and backup folders.
- Added automatic symbol-to-footprint linking.
- Added KiCad project library table updates.
- Added downloads folder watcher for automatic imports.
- Added provider search links.
- Added Windows installer build using PyInstaller and Inno Setup.
