What it handles

The app is focused on keeping third-party CAD downloads tidy inside each KiCad project.

Symbols and footprints

Merges imported symbols and footprints into configured project libraries.

3D models and metadata

Copies models, datasheets, source ZIPs, and import metadata into organized folders.

Automatic imports

Can watch a downloads folder and import new component ZIP files when they arrive.

First use with KiCad 10

KiCad 10 can cache project library tables while a project is open, so first setup works best before KiCad loads the project.

Create or choose your KiCad project

The project folder should contain a .kicad_pro file.

Close KiCad fully

Close the schematic editor, PCB editor, and KiCad project window before the first setup.

Configure the importer

Select the KiCad project root, name your components library, select your downloads folder, and save.

Open KiCad and import

After the library exists and KiCad has loaded it, new imports go into the same library.
